# DocumentPath

ドキュメントのファイルシステムパスを表す値オブジェクト。

## 属性

### value

パス文字列。

- スラッシュ区切りの階層パス
- 相対パスまたは絶対パス
- ファイル拡張子を含む

## ビジネスルール

- 空文字列は許可されない
- パス区切り文字は統一してスラッシュ（/）を使用する
- 不正なファイル名文字（<, >, :, ", |, ?, *, \）は使用不可
- パス長は最大1000文字まで
- ディレクトリトラバーサル攻撃を防ぐため「../」「./」の連続は検証する
- 大文字小文字を区別する
- 末尾のスラッシュは正規化時に除去される
